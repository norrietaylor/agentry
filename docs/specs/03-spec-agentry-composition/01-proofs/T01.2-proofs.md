# T01.2 Proof Summary: Implement DAG validation at parse time

## Task
Add `@model_validator(mode="after")` to `CompositionBlock` in `src/agentry/models/composition.py`
that validates the composition DAG structure at parse time.

## Implementation
- Added `validate_dag` model validator to `CompositionBlock`
- Uses `graphlib.TopologicalSorter` (Python 3.9+ stdlib) to detect cycles
- Validates all `depends_on` references resolve to existing node IDs
- Validates all `inputs` source expressions match `<node_id>.output` or `<node_id>.output.<field>`
- Validates that input source node IDs are listed in `depends_on`
- Produces clear error messages identifying cycle paths (e.g. "Cycle detected: A -> B -> A")
