- Uses `graphlib.TopologicalSorter` incremental API (`prepare()`, `get_ready()`, `done()`) for async dispatch.

- Runner teardown in `finally` block for each node.
- Produces `CompositionRecord` with per-node statuses, overall status, and wall-clock timing.
- Saves `CompositionRecord` to `<run_dir>/composition-record.json`.
