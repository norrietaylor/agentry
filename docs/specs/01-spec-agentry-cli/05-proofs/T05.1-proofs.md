# T05.1 Proof Artifacts

## Task: Layer 1 (Schema Validation) and Layer 2 (Side-Effect Allowlist)

## Summary

Implemented the output validation pipeline's first two layers:

- **Layer 1** (`src/agentry/validation/layer1.py`): Validates agent output against
  the JSON Schema declared in `output.schema` using the `jsonschema` library.
  Reports `schema_path`, `failed_keyword`, and a human-readable `message` on failure.

- **Layer 2** (`src/agentry/validation/layer2.py`): Checks each non-read-only tool
  invocation against the `output.side_effects` allowlist. Blocks and reports
  undeclared side effects with the `side_effect` identifier, `allowlist`, and
  remediation `message`.

- **Result types** (`src/agentry/validation/result.py`): `LayerResult` and
  `ValidationResult` dataclasses with `to_dict()` for JSON serialisation into
  execution records.

- **Exceptions** (`src/agentry/validation/exceptions.py`): `SchemaValidationError`
  and `UndeclaredSideEffectError` for explicit error signalling.
