# T05.2 Proof Artifacts

## Task: Layer 3 (Output Path Enforcement) and Pipeline Orchestration

## Summary

Implemented the output validation pipeline's third layer and the full
sequential pipeline orchestrator, plus budget enforcement:

- **Layer 3** (`src/agentry/validation/layer3.py`): Verifies all file writes
  target paths within the declared `output.output_paths` list. Uses prefix
  matching: a declared path `.agentry/runs/` allows any write whose path
  starts with that prefix. Blocks and reports undeclared writes with the
  `path`, `output_paths`, and `message` fields.

- **Pipeline orchestrator** (`src/agentry/validation/pipeline.py`):
  `run_pipeline()` executes Layers 1, 2, and 3 in sequence, halting on the
  first failure. Returns a `ValidationResult` with `validation_status` and
  `layer_results` for every evaluated layer.

- **Budget enforcement** (`src/agentry/validation/pipeline.py`):
  `apply_budget()` enforces `max_findings` truncation on the output
  `findings` list. Appends a `_truncation_note` key when truncation occurs,
  describing how many findings were removed. Returns a `BudgetResult`
  dataclass with truncation metadata.

- **Exception** (`src/agentry/validation/exceptions.py`):
  `UndeclaredOutputPathError` for Layer 3 path violations.

- **Public API** (`src/agentry/validation/__init__.py`): Updated to export
  `validate_output_paths`, `run_pipeline`, `apply_budget`, `BudgetResult`,
  and `UndeclaredOutputPathError`.
