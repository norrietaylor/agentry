3. LocalBinder class implementing the protocol — implemented.

   NotAGitRepositoryError with clear message for non-git directories.

- Tool implementations (repository:read, shell:execute) deferred to T03.3
  (skeleton binds stubs that raise NotImplementedError on call).
- map_outputs() returns paths under .agentry/runs/<run_id>/ as specified.
