## Implementation Summary
- Created `src/agentry/executor.py` with:
  - `AgentExecutor` class with `run()` method as the main entry point
  - Multi-turn conversation loop supporting tool invocations (max 20 turns)
  - Retry logic with exponential backoff via `RetryConfig`
  - Per-execution timeout enforcement via asyncio + LLMConfig.timeout
  - `ExecutionRecord` dataclass recording tokens, timing, tool invocations, retries
  - `format_inputs_as_messages()` for converting resolved inputs to LLM messages
  - JSON output parsing with code fence extraction
  - Tool name mapping (internal `__` to original `:` format)
- Created `tests/unit/test_agent_executor.py` with 56 comprehensive tests
- All 484 unit tests pass (including 56 new + 428 existing)
- Zero ruff lint warnings
