## Changes Made

Modified `src/agentry/binders/local.py`:

1. Added `import re` to imports.
2. Added `_GIT_REF_PATTERNS` module-level constant: a tuple of compiled regex patterns covering HEAD~N/^N, 7-40 hex SHA prefixes, range syntax (..  /  ...), and named refs like `main`, `origin/main`.
3. Added `_is_git_ref(value: str) -> bool` helper function that returns `True` when the value matches a git ref pattern, `False` when it looks like raw diff text (multi-line, empty, or containing special diff characters).
4. Rewrote `_resolve_git_diff()` to implement the following logic:
   - If `_is_git_ref(value)` is `False` → return `value` as-is (raw diff text pass-through, no git command executed).
   - If `_is_git_ref(value)` is `True` → validate target is a git repo (raise `NotAGitRepositoryError` if not), then run `git diff <value>` and return stdout.
   - If `git diff` raises `CalledProcessError` → fall back to returning `value` as-is.
