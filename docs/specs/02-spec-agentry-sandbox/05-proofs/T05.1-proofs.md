## Key Design Decisions

- Private key is stored outside the project directory (`~/.agentry/`) to prevent accidental commits.
- Private key file is created with mode `0o600` (owner read/write only).
- Private key directory is set to `0o700` (best-effort, non-fatal on unsupported platforms).
- Public key is stored in `.agentry/public-key.pem` inside the project so it can be committed to version control.
- The CLI output explicitly instructs the user to `git add` the public key and warns never to commit the private key.
