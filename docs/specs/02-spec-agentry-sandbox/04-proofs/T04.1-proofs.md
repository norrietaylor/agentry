**File created:** `src/agentry/security/preflight.py`

## Requirements Implemented

- `PreflightResult` dataclass with `passed`, `check_name`, `message`, `remediation` fields
- `PreflightChecker` class with pluggable check interface (accepts any object satisfying `PreflightCheck` protocol)
- `PreflightChecker.run_all()` executes all registered checks and returns a list of `PreflightResult`
- All failures collected together (not one-at-a-time) — full run even when early checks fail
- `skip_preflight=True` bypasses all checks with a warning log (corresponds to `--skip-preflight` CLI flag)
- `PreflightFailedError` exception that contains all failures when `raise_on_failure=True`
- `to_manifest_entries()` serialises results for setup manifest recording
- `report_failures()` formats human-readable failure output
- `any_failed()`, `failures()` convenience helpers
- `add_check()` for dynamic registration of checks
