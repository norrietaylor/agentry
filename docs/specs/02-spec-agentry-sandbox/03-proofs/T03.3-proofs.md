# T03.3 Proof Summary

## Task
Implement 'agentry setup' CLI command and integrate with 'agentry run'

## Implementation

### Files Modified
- `src/agentry/cli.py` - Primary implementation file
- `tests/unit/test_cli.py` - Updated and added tests

### Changes

**`src/agentry/cli.py`**:
1. Added `_MinimalRunner` shim class: satisfies `SetupPhase`'s `RunnerProtocol` interface
   (no-arg `provision() -> dict`) without starting real containers.
2. Replaced stub `setup` command (printed "Not yet implemented") with full implementation:
   - Accepts `WORKFLOW_PATH` positional argument
   - Accepts `--skip-preflight` flag for development use
   - Loads workflow via `load_workflow_file()`
   - Builds preflight checks: `AnthropicAPIKeyCheck`, `DockerAvailableCheck`, `FilesystemMountsCheck`
   - Instantiates `SetupPhase` with `_MinimalRunner`
   - On success: prints manifest path (text format) or emits JSON with `status: ok`
   - On preflight failure: emits diagnostic with `check_name`, `message`, `remediation`
3. Integrated setup phase into `agentry run`:
   - Before calling executor, loads workflow and runs setup phase
   - Preflight failure aborts run with exit code 1 and diagnostic
   - If setup components unavailable: logs debug and falls through gracefully

**`tests/unit/test_cli.py`**:
1. Updated `test_setup_not_yet_implemented` -> `test_setup_requires_workflow_path`
2. Added 7 new tests covering setup command behavior

## Key Behaviors Verified

1. `agentry setup WORKFLOW_PATH --skip-preflight` exits 0 and emits manifest path
2. JSON output format (`--output-format json`) emits `{status: ok, manifest_path: ...}`
3. Manifest file is created on disk at `.agentry/runs/<timestamp>/setup-manifest.json`
4. Manifest contains all required fields: workflow_name, workflow_version, container_image,
   filesystem, network, resources, sandbox_tier, timestamp
5. Manifest uses credential fingerprints (SHA-256), not raw API key values
6. `agentry setup` without WORKFLOW_PATH exits non-zero (usage error)
7. `agentry setup nonexistent.yaml` exits 1 with "not found" message
8. Full test suite passes with no regressions (969 passed, 1 skipped)
