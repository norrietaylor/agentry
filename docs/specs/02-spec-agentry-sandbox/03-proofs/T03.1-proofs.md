## Key Behaviors Tested

1. **Tool stripping**: `strip_tools()` filters available tools to manifest-only set
2. **Abort on strip**: `abort_on_strip=True` raises `ToolManifestViolationError`
3. **Preflight failures**: Failing checks raise `PreflightError` before agent runs
4. **First-failure stops**: Subsequent preflight checks are not run after failure
5. **Runner lifecycle**: `provision()` before execution, `teardown()` in finally block
6. **Teardown on failure**: Runner teardown runs even on preflight/executor failure
7. **Output validation**: Three-layer pipeline runs on structured JSON output
8. **Validation skipped**: No validation when output is plain text or executor errored
9. **Schema validation failure**: Captured correctly in result
10. **Exception models**: Proper fields on ToolManifestViolationError and PreflightError
11. **Protocol compliance**: MockRunner satisfies RunnerProtocol
