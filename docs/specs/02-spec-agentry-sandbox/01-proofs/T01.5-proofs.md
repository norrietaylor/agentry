## Implementation Summary

### DockerRunner.execute()
- Writes agent config JSON to temp file on host
- Copies config into container at /config/agent_config.json via put_archive
- Starts the container (which runs the shim command set during provision)
- Waits for container completion with configurable timeout
- On timeout: kills container with SIGKILL, returns exit_code=137, timed_out=True
- Collects stdout/stderr from container logs
- Reads result.json from the output directory
- Returns ExecutionResult with all metadata

### Runtime Shim (src/agentry/runners/shim.py)
- Reads JSON config from mounted path (default: /config/agent_config.json)
- Validates required keys: system_prompt, resolved_inputs, tool_names, llm_config
- Constructs AgentExecutor and runs the agent
- Writes execution record to /output/result.json
- Exit codes: 0 (success), 1 (agent error), 2 (shim startup error)

- Container command now set at provision time to run the shim automatically
